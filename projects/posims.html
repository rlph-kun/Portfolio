JohnTech POS & Inventory (Ongoing)

Full-stack POS with role-based auth, real-time sales & inventory tracking for two branches, and automated management tools.

HTML CSS JavaScript PHP MySQL

Overview

The system centralizes sales, inventory, and branch operations. It supports cashier and admin roles, synchronous stock updates, sales history, and daily sales reports.

Key Features

  • Role-based access: Admin, Cashier
  • Real-time inventory synchronization across two branches
  • Audit log and quick checkout
  • Daily, weekly, and monthly sales reporting
  • Supplier and purchase order management

Architecture

Built with PHP and MySQL on a classic MVC structure, enhanced with vanilla JavaScript for interactivity. Deployed on a shared hosting environment with database backups.
